## 🚀 部署步骤（请按顺序执行）

### 第一步：安装Node.js
如果还没有安装，请先下载安装：
https://nodejs.org/zh-cn/download/

### 第二步：打开命令提示符（CMD）
按 `Win + R`，输入 `cmd`，回车

### 第三步：进入项目目录
```bash
cd D:\profiles-vscode\travel
```

### 第四步：安装依赖
```bash
npm install
```

### 第五步：初始化数据库
```bash
npm run init-db
```

### 第六步：启动服务器
```bash
npm start
```

### 第七步：访问应用

**用户端：** http://localhost:8080

**管理员后台：** http://localhost:8080/admin.html

## 🔧 常见问题

### Q: npm install 失败？
A: 请尝试以下命令：
```bash
# 清除npm缓存
npm cache clean --force

# 重新安装
npm install
```

### Q: 端口被占用？
A: 修改server.js中的端口号，然后重启

### Q: 数据库错误？
A: 删除travel.db，重新运行：
```bash
npm run init-db
```
