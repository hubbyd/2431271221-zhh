# 🌍 GitHub Pages 部署指南

## 快速设置步骤

### 第一步：打开GitHub仓库设置
请在浏览器中打开以下链接：
👉 https://github.com/hubbyd/2431271221-zhh/settings/pages

### 第二步：配置GitHub Pages
在打开的页面中，按照以下步骤操作：

1. **Source（来源）** 部分
   - 点击下拉菜单，选择 **"Deploy from a branch"**

2. **Branch（分支）** 部分
   - 点击下拉菜单，选择 **"main"**
   - 旁边的文件夹选择 **"/ (root)"**

3. **点击 Save（保存）按钮**

### 第三步：等待部署
- 等待约 **1-2分钟**
- 页面会刷新，您会看到绿色的提示框
- 显示：`Your site is live at https://hubbyd.github.io/2431271221-zhh/`

### 第四步：访问您的网站
- 点击上面的链接访问您的AI旅行助手
- 或者访问：`https://hubbyd.github.io/2431271221-zhh`

## 🔧 自定义域名（可选）

如果您有自定义域名，可以：
1. 在 Settings → Pages → Custom domain 输入您的域名
2. 配置DNS记录指向GitHub
3. 等待SSL证书自动配置
